Upload 2025 Sales Orders PDF
Upload a PDF file containing sales order data. The system will attempt to extract table data.
Expected Format

The PDF should contain tables with the following columns:

  • Order Number - Unique order identifier
  • Date - Order date (MM/DD/YYYY or YYYY-MM-DD)
  • Customer - Facility/company name
  • SKU - Product SKU (14Fr, 16Fr, 18Fr, etc.)
  • Quantity - Number of units
  • Lot Number (optional) - Lot identifier
Import Notes
  • Sales orders will be created with source="pdf_import"
  • Customers will be auto-created or matched by facility name
  • Duplicate orders (same order number + date) will be skipped
  • Order lines will be created for each SKU/quantity combination
  • Parse errors will be reported but won't stop the import
